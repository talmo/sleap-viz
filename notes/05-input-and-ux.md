# Input Handling & UX

## Desktop MVP Controls

### Keyboard
- `Space` — play/pause
- `J` / `K` or `←` / `→` — step backward/forward; `Shift` = ×10; `Ctrl/Cmd+Shift` = ×100
- `Ctrl/Cmd` + `+` / `-` — zoom in/out

### Mouse/Trackpad
- **Timeline drag (no mouse-up)** — scrub continuously
- **Scroll wheel** / **pinch** — smooth zoom
- **Click+drag** — pan
